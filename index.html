How to use the demo

  • Click Launch the live demo above (or any card) and sign in — password is demo for every account, or use the one-click persona buttons.
  • See tenant isolation: sign in as Anna (Client User, Acme) → you see only Acme's tickets. Sign out, sign in as Sara (System Admin) → you see every company.
  • Run the workflow: raise a ticket → it appears in the queue → assign an agent (as System Admin) → start work / comment / resolve → watch the history and dashboard counts update.
  • Changes persist in your browser. Use ↺ Reset demo (top-right) to restore the seed data.
  • ⚠️ Simulated client-side login — not real security. Real auth + isolation are built in Oracle APEX.
